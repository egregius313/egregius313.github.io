NAME

ed -- the standard text editor

SYNOPSIS

ed [-r] minnix [-3]

DESCRIPTION

ed is a developer as well as a course assistant and cybersecurity student at Stevens Institute of Technology.

He is fond of free and open source software such as emacs(1), firefox(1), and python(1). He is experienced with writing software in several programming languages, including Python, C, C#, Java, OCaml, and several Lisps.

ed is interested in both low and high level systems. He enjoys studying how compilers generate code and how code is executed at the Assembly level. He also enjoys designing code using mathematical and higher level constructs (insert monad joke here).

LIMITATIONS

ed can never seem to be content with his emacs(1) configuration.

REGULAR EXPRESSIONS

Things which ed can regularly be heard saying include `just use emacs(1)', `I once wrote that in 2 lines of awk(1)', and `the math is not that hard'.

SEE ALSO

mail(1), git(1), resume(4), linkedin(4), menu(3x), init.el(4)

HISTORY

The first version of ed appeared in 1946. The current version appeared in 1998.

BUGS

ed starts to perform very poorly when using solely Windows for an extended period of time.
